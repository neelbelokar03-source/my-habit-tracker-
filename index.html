One-time Firebase Setup
🔥 Connect Firebase (Free · 2 mins)
Do this once — your data will then sync across all devices automatically.
1
Go to console.firebase.google.com and create a free project.
2
Inside the project click Add app → Web. Copy the firebaseConfig object it gives you.
3
Go to Authentication → Sign-in method → Google — enable it.
4
Go to Firestore Database → Create database → choose Test mode.
5
Open this HTML file in a text editor. Find the firebaseConfig section and paste your values. Save and reopen.
💡 Firebase free tier = 1 GB storage + unlimited logins. More than enough for personal use forever.
